# 🔔 줌 수업 학교종 타이머 (classbell_timer)

정해진 시각에 학교 종소리가 자동으로 울리는 온라인 타이머. 외부 파일 없이 종소리 mp3 12개가 페이지에 내장돼 있습니다.

## ▶ 바로 실행 (이 링크를 열면 됩니다)

### 👉 https://sumilee-pcu.github.io/classbell_timer/

## 줌 수업에서 쓰는 법
1. 수업 전 위 링크를 크롬 탭으로 열고 **▶ 타이머 시작** 클릭.
2. 줌 **화면 공유 → "소리 공유(Share sound)"** 체크.
3. 탭을 닫지 말고, 맥 절전(슬립)을 꺼두세요.

## 기능
- 정해진 시각마다 학교 종소리 자동 재생 (12개 중 랜덤)
- 실제 종소리 mp3 ↔ 전자 합성음(웨스트민스터 차임) 전환
- 시간표 추가/삭제, 50분 수업 프리셋, 브라우저에 저장

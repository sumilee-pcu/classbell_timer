# 🔔 줌 수업 학교종 타이머

온라인 줌 강의 중 **정해진 시각에 학교 종소리가 자동으로 울리는 타이머**입니다.
9:00 수업 시작, 9:50 쉬는 시간, 10:00 다시 시작 … 처럼 시간표대로 종이 울립니다.
종소리 mp3 12개가 페이지에 내장되어 있어 **외부 파일 없이 어디서나** 동작합니다.

---

## ▶ 바로 실행하기 (클릭)

## 👉 **https://sumilee-pcu.github.io/classbell_timer/**

위 링크를 클릭하면 설치 없이 브라우저에서 바로 실행됩니다. (PC·스마트폰 모두 가능)

---

## 📺 줌 수업에서 학생들에게 들려주는 법

1. 수업 전, 위 링크를 **크롬 탭으로 열어**둡니다.
2. 페이지에서 **`▶ 타이머 시작`** 버튼을 한 번 클릭합니다.
   *(브라우저는 사용자가 한 번 클릭해야 소리 재생을 허용합니다.)*
3. 줌에서 **화면 공유** → 공유 창 왼쪽 아래 **`소리 공유(Share sound)`** 를 체크합니다.
   *(이걸 켜야 내 컴퓨터 종소리가 학생들에게 전달됩니다.)*
4. 수업 내내 탭을 닫지 말고, **맥/PC 절전(슬립)을 꺼둡니다.**
   *(화면이 잠들면 소리도 멈출 수 있습니다.)*

---

## ✨ 기능

- ⏰ **시각마다 자동 종소리** — 설정한 시간이 되면 자동 재생
- 🎲 **실제 종소리 12개 랜덤 재생** — 매번 다른 종소리 (직전 곡 연속 방지)
- 🎹 **전자 합성음 모드** — 웨스트민스터 차임(빅벤) 멜로디로 전환 가능
- 📝 **시간표 편집** — 시각·이름 추가/삭제, `50분 수업 프리셋` 제공
- 🔁 **이전 소리 자동 정지** — 새 종이 울리면 이전 소리는 즉시 끊김
- 💾 **자동 저장** — 입력한 시간표는 브라우저에 저장되어 다음에 열어도 유지

## 💡 팁

- 처음 열 때 종소리 데이터(약 2.4MB)를 받느라 1~2초 걸릴 수 있고, 이후엔 캐시되어 빠릅니다.
- 맥에서 절전 끄기: 터미널에 `caffeinate -d` 를 켜두거나 시스템 설정에서 디스플레이 끄기 시간을 늘리세요.
- 즐겨찾기(북마크)에 위 링크를 추가해두면 매번 클릭 한 번으로 사용할 수 있습니다.
